## moozza
Здесь представлено описание проекта сайта Moozza, реализованном на Node-Red. 

Это сайт для учебной организации и/или какой-либо закрытой фирмы. Веб-приложение разворачивается в локальной сети, для развёртывания в глобальной сети требуется хостинг.

Проект большой, поэтому подробное описание займёт большоое количество времени и места, здесь кратко опишу каждый из потоков, который выполняет свою задачу.

Для веб-приложения требуются дополнительные модули Node-Red, такие как node-red-dashboard, node-red-contrib-bcrypt и node-red-node-sqlite.

Также веб-приложению необходим файл .db с таблицами, которые создаются в каждом потоке. Здесь он есть для примера, в программе прописан его адрес: C:\sqlite\myusersdb.db

Файл проекта целиком в форматированном виде json --> flowmoozza.json. 

Файл потока "Авторизация", который осуществляет работу с пользователями --> auth.json.

Файл потока "Предметы", который осуществляет работу с предметами и лабораторными работами по этим предметам --> subj.json.

Файл потока "Чаты", который реализует страницу чатов по разным предметам --> mess.json.

Файл потока "Главная/Профиль", который реализует страницу "Главная" из бокового меню, где пользоователь может изменить свои данные --> main.json.

Файл потока "Расписание", который реализует расписание занятий с различными категориями, преподавателями и в различные временные отрезки --> calen.json.
